# 历史

LuaSTG/LuaSTGPlus具备多个衍生版本，此处进行了整理供使用者捋清各个版本之间的关系。

版本名称 作者 平台 引擎 图形API 脚本引擎
LuaSTG (opens new window) 隔壁的桌子 Windows HGE DirectX 8 Lua5.1
LuaSTGPlus (opens new window) 9chu (opens new window) Windows fancy2d DirectX 9 LuaJIT
LuaSTGPlus-v2 (opens new window) 9chu (opens new window) Windows/MacOS/Linux/HTML5 D3D11/D3D12/Vulkan/OpenGL LuaJIT/Lua5.1
LuaSTGExPlus (opens new window) Xiliusha (opens new window) Windows fancy2d DirectX 9 LuaJIT
LuaSTG-x (opens new window) Xrysnow (opens new window) Windows/MacOS/Linux/Android/IOS cocos2d-x OpenGL/Vulkan LuaJIT
LuaSTG-Sub (opens new window) Demonese (opens new window) Windows D3D11 LuaJIT

各版本时间线参考了 @Demonese (opens new window) 整理的资料。
