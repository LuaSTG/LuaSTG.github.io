# 调试控制台

当运行在开发模式下，可以通过~键唤出调试控制台。

调试控制台可以显示当前的日志信息。如果系统打印出 Error 级别日志，调试控制台也会自动弹出。

你可以在调试控制台的命令行中执行任意 Lua 语句，同时控制台会显示语句的返回值。

通过在日志文本框中单击鼠标右键还可以切出其他子项调试窗口。
